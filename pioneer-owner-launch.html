Pioneer NFT owner setup.

This page uses your connected owner wallet to prepare the already deployed Pioneer contract. It never asks for a private key. The backend signer private key stays only on the downloads/faucet server.

Order: backend online at pioneer.lustchain.org → set mint authorizer → set metadata → freeze metadata → fund 10,000 LST → start mint. Do not freeze before the metadata URL is working.

FINAL9 rules:
- One active public reward wallet = one mining chance.
- More CPU/GPU does not multiply chances.
- Public miner block rewards go to the public reward wallet.
- PROD1/PROD2 have no wallet privilege in FINAL9.
- Official producer rewards use the Treasury path.
- After Active, wait about 64 blocks before the miner fully enters rotation.
- The maturity period helps protect the network against instant wallet rotation and spam registrations.

Package status:
- Windows package includes full live status.
- Ubuntu package includes START, STATUS, LIVE STATUS, STOP and RESET scripts.

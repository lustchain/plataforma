TX-FEED fairness

Keep public transactions inside your miner.

TX-FEED checks the public pending transaction feed and injects raw pending transactions into your local miner node. This gives public miners a fair chance to include transactions when they mine a block.

Security

Never enter your seed.

The official site only asks MetaMask to sign/send a registration transaction. Never type your seed phrase on any website.

Wallet rule

One machine, one wallet.

For clean testing and production, use one registered wallet per miner machine. Do not run the same wallet on many devices at the same time.

Safe start + TX-FEED

No isolated mining.

The miner starts with mining off, waits for peers, chain ID and gap, then keeps TX-FEED active so pending public transactions can reach your local node.

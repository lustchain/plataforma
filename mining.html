Register and activate

Use the same wallet

The wallet used to register/activate must be the same public wallet pasted into the miner.

1. Connect wallet

Open the register page, connect MetaMask and switch/add LUST Chain.

2. Send Register / Active

Confirm the transaction. This tells the chain your wallet wants to participate in public mining.

3. Start miner

After confirmation, run the miner package and paste the same public wallet address.

Open Register / Active Open explorer
Windows

Windows steps

Extract ZIP

Extract the H2K ZIP to a clean folder. Do not reuse an old miner folder.

Register wallet

Run REGISTER-MINER-WINDOWS.bat or use the button on this page.

Start miner

Run START-LUST-MINER-WINDOWS.bat and paste only your public 0x wallet.

Check status

Run STATUS-LUST-MINER-WINDOWS.bat. Wait until it shows SYNCED / MINING ACTIVE.

Stop if needed

Run STOP-LUST-MINER-WINDOWS.bat to stop the miner.

Wrong chain?

If Chain ID is not 0x1b0b, stop and use the H2K package from this page.

Ubuntu steps

Install and start

Extract the package, enter the folder and run the start script.

tar -xzf lust-v31h2k-miner-ubuntu-FULL.tar.gz
cd LUST_V31H2K_UBUNTU_FULL

Status and stop

Use the status script to check sync and mining. Use stop when needed.

./status-lust-miner-ubuntu.sh
./stop-lust-miner-ubuntu.sh
Shares and rewards

Simple reward rules

1 wallet = 1 chance

Mining chance is per registered/active wallet. More CPU/GPU power does not multiply shares.

One public miner

If only one public miner is active and produces the block, it receives 2 LST plus transaction fees.

Two or more public miners

The producer receives 70% = 1.4 LST. Other active public miners share 30% = 0.6 LST.

Fees

Transaction fees go to the block producer. Fees are not shared in the passive 30%.

No public miners

prod-1/prod-2 keep the network alive only as emergency producers.

Stay active

If your miner/wallet is offline or not active, it may stop participating until you activate again.

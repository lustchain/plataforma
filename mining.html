Run checklist

The miner is ready only when all checks pass.

Before telling a community member that mining is working, confirm these fields in the local miner control center.

Sync

Gap 0 and not syncing.

The local block must match the public block and Syncing must be False.

Peers

Peers 1 or more.

For production, more peers is better. If peers stay 0, restart the miner or check firewall/router.

Canonical safety

Hash check is mandatory.

The current Windows V31 package shows Local hash, Public hash, Compare height and Canonical YES/NO. Gap 0 alone is not enough.

If you change wallet on a PC after mining opens: stop/start the miner with the new wallet, wait for READY + Canonical YES, then Register the new wallet and send a fresh Activate / Heartbeat. The old wallet may still appear in rolling live stats for a short time until the window rolls out.

Reward rules

Exactly how each 2 LST block reward works.

The V31 reward model uses a fixed 2 LST block reward and a 70/30 split when two or more public miners are active.

0 public miners

Emergency official fallback.

Official producer keeps the chain alive and receives the full 2 LST only when no public miner is active.

1 public miner

One public miner receives 2 LST.

If there is no other public committee member, the active public producer receives the full block reward.

2+ public miners

70% producer · 30% public committee.

The producer receives 1.4 LST. The other active public miners share 0.6 LST. Official/prod wallets receive none while public miners are active.

Reward accounting is designed around a fixed 2 LST block reward. When two or more public miners are active, the producer receives 70% and the public committee share receives 30%.

Troubleshooting

Common problems and what they mean.

Use this guide after public mining is active.

Gap is not 0

Miner is still syncing.

Wait. If the gap does not reduce, restart the miner. If still stuck, delete only the local miner data folder on that PC, not the wallet file.

Peers are 0

Network connectivity problem.

Check firewall, VPN, router and internet. Bootnodes must be reachable on P2P.

Register fails

Usually wrong network or no gas.

After public mining is active, switch MetaMask to LUST Chain and claim faucet LST before trying again. Register and Heartbeat need native LST gas.

Official fallback blocks

Check the window.

When public miners are active, official fallback blocks should decrease in the recent window. If public miners are offline, fallback production appears to keep the chain alive.

Already registered wallet

Use Active only.

If this wallet already registered before, do not register again. Start the miner, wait for READY/Canonical YES, then send Active / Heartbeat.

Wrong package

Delete old folders.

If the EXE SHA is not 947e6353..., delete the file/folder and download the V31H2E binary again from this page. Any 258eb627... H2D EXE is blocked for public mining.

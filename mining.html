Windows flow

Clean test steps

Do not reuse old folders. Use one wallet and one clean folder for each test.

Download fresh ZIP

Download the Windows ZIP with the stable snapshot cache-buster and extract to a clean folder, for example C:\LUST-Miner-H2J.

Start miner

Run START-LUST-MINER-WINDOWS.bat and paste only the public 0x wallet. Never paste seed phrase or private key.

Confirm Chain ID

The terminal must say OK: local node is LUST Chain 0x1b0b. If it shows 0x1, stop.

Wait for sync

Gap can be high at first. Let it fall naturally. The stable package should not reinstall old snapshots because of gap.

Open Status

Run STATUS-LUST-MINER-WINDOWS.bat. Good status: Chain ID 0x1b0b, peers above 0, local block not 0.

Check rewards later

Rewards are confirmed by wallet balance/RPC. The explorer can show native custom reward as 0 in some blocks.

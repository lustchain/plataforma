Simple setup

How to start mining

Follow this order exactly. Do not register again if your wallet already uses the same Operator Address.

Download FINAL6

Download the official Windows ZIP and extract it into a new folder.

Download Windows FINAL6

Start miner

Run START-LUST-MINER-WINDOWS.bat. Paste your public reward wallet when asked. The miner syncs first and only starts mining after Gap 0.

Wait for active

Wait until the console shows SYNCED / MINING ACTIVE. FINAL6 also starts the official RPC transaction puller automatically.

Copy Operator

Copy the Operator Address shown by the miner. It is not your private key. Never paste a seed phrase anywhere.

Register

Connect the same reward wallet here, paste the Operator Address and click Register Wallet + Operator. This links your wallet to the local block signer.

Active / Heartbeat

After Register confirms, click Active / Heartbeat. Keep the miner running and use STATUS-LUST-MINER-WINDOWS.bat to watch blocks and balance.

Miner files

What each button does

START

START-LUST-MINER-WINDOWS.bat starts sync-first mining. Mining starts only after the node reaches the public chain.

STATUS

STATUS-LUST-MINER-WINDOWS.bat shows wallet, operator, gap, peers, blocks mined, balance and session gain.

TX relay

START-LUST-TX-RELAY-WINDOWS.bat can manually start the official transaction puller if needed.

STOP

STOP-LUST-MINER-WINDOWS.bat stops local mining processes. Do not stop it while you want to mine.

Shares and rewards

How mining rewards work

One wallet = one chance

Each registered and active wallet has one mining chance/share. More CPU or GPU power does not multiply the chance.

Solo public miner

If only one public miner is active and producing blocks, that miner can receive the public mining reward for the blocks it produces, plus transaction fees.

Multiple public miners

The block producer receives the producer share and the shared portion is distributed by protocol rules to eligible active public participants. The status tool shows estimates; wallet balance and explorer history are the final source.

Emergency producers: prod-1/prod-2 are fallback producers only. When public miners are healthy, the guard keeps prod-1 inactive and public miners produce the blocks.
Testing more miners

How to test a second miner

Recommended

Use another PC or VPS with a different reward wallet. Download FINAL6, start it, copy the new Operator Address, register that wallet, then activate it.

Same PC warning

The standard Windows package uses fixed local ports. Running multiple miners on the same PC requires a special multi-port test package; do not open two normal FINAL6 miners on the same Windows user.

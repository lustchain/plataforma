Test with a fresh folder and the H2J ZIP only.

Do not reuse the old H2I folder. The package asks only for your public 0x wallet address. Never enter seed phrase or private key.

Delete old folder

Rename/delete old LUST-Miner, H2I, H2E, H2D, V31 or V7A30 folders. Use a clean folder.

Download H2J ZIP

Download lust-v31h2j-miner-windows-FULL.zip from this page only.

Extract All

Right-click the ZIP and choose Extract All. Do not run files directly from inside the ZIP preview.

Verify binary

Run VERIFY-H2J-BINARY-WINDOWS.bat. The EXE SHA must match the value shown below.

Open register helper

Run REGISTER-MINER-WINDOWS.bat. It opens the official registration page in your browser.

MetaMask actions

Connect wallet, switch to LUST Chain, claim faucet gas if needed, then confirm Register wallet and Activate / Heartbeat.

Start miner

Run START-LUST-MINER-WINDOWS.bat. Paste only your public 0x reward wallet.

Check status

Run STATUS-LUST-MINER-WINDOWS.bat. Continue only with peers > 0, Syncing False, Canonical YES and Gap low/0.

Confirm on-chain

If the miner says it found something, confirm by public RPC/explorer and wallet balance. Do not trust local text alone.

Reward and fairness rules

One wallet, one deterministic chance.

CPU/GPU power does not increase chance. The registered active wallet is what matters.

No public miners active

Official fallback only

Official producers keep the chain alive and the producer receives the normal 2 LST block reward.

One public miner active

Public first chance

The public miner has first deterministic chance. If official fallback produces, official receives 1.4 LST and the active public miner receives 0.6 LST.

Two or more public miners

70% / 30%

The block producer receives 1.4 LST and other active public miners share 0.6 LST. Official wallets are excluded from the passive 30% when public miners are active.

Explorer note: Blockscout may show Reward 0 LST for custom native rewards. Confirm reward by wallet balance, RPC audit and canonical blocks.

Troubleshooting

Fix the common test problems quickly.

Do not improvise with old packages. Restart cleanly from the official H2J ZIP.

Gap increases

Stop and resync cleanly.

Use STATUS first. If gap keeps increasing, stop miner, make sure it is H2J, delete only local chain data if instructed, then start from latest snapshot.

Peers are 0

Network or firewall issue.

Wait after restart. If it stays 0, check VPN, firewall, router and whether P2P connections are blocked.

Register fails

Wrong network or no gas.

Switch MetaMask to LUST Chain and claim faucet LST. Register/Heartbeat require tiny gas, not a deposit.

Already registered

Use Active / Heartbeat.

Do not register repeatedly. Start the H2J miner, then send Active / Heartbeat again from the same wallet.

Local says found block

Confirm on public chain.

Only count it if the block exists on public RPC/explorer and the wallet balance/reward audit confirms it.

Wrong package

Delete old folder.

Do not use V7A30, V31, V31H2C, V31H2D, V31H2E or V31H2I folders. Use V31H2J only.

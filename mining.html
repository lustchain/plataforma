First-time setup

Start mining in 6 steps

Follow the steps in order. The miner syncs first and starts mining only when it is ready.

Download

Download the official Windows miner ZIP and extract it to a normal folder such as Downloads or Desktop.

Download Windows ZIP

Start

Run START-LUST-MINER-WINDOWS.bat. Paste your public reward wallet address when asked.

Wait for sync

Wait until the miner shows Gap 0 and SYNCED / MINING ACTIVE.

Copy Operator

Copy the Operator Address shown in the miner window. This address signs blocks locally.

Register

Connect the same reward wallet here, paste the Operator Address, then click Register Wallet + Operator.

Activate

After Register confirms, click Active / Heartbeat. Keep the miner running and use Status to monitor rewards.

Rewards

How LST mining rewards are shown

One wallet, one chance

Each registered and active reward wallet has one public mining chance.

Block producer

The miner that produces a block receives the producer side and transaction fees for that block.

Shared rewards

The protocol handles the active public miner share. The final source of truth is your wallet balance and the explorer.

Status is practical: the live status shows your mined blocks, current balance and real session gain instead of fake fixed reward numbers.

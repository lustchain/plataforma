Simple flow

How to start mining

Download H3F

Download the official Windows H3F ZIP and extract it into a new folder.

Download Windows H3F

Start miner

Run START-LUST-MINER-WINDOWS.bat, paste your public reward wallet, and wait for SYNCED / MINING ACTIVE.

Copy Operator

Copy the Operator Address shown by the H3F miner. Do not use your private key. Do not use old H3B/H3D register files.

Register

Connect the same reward wallet here, paste the Operator Address, and click Register Wallet + Operator.

Active / Heartbeat

After register confirms, click Active / Heartbeat. Use the faucet if the wallet needs gas.

Open faucet

Keep it running

Keep the H3F miner open. The status tool should show peers above 0, gap 0, and SYNCED / MINING ACTIVE.

Shares and rewards

How rewards work

One wallet = one chance

Each registered and active wallet has one mining chance/share. Stronger CPU or GPU does not multiply your chance.

One public miner

If only one public miner is active and produces the block, that miner receives 2 LST plus transaction fees.

Two or more publics

The selected block producer receives 70% = 1.4 LST. The remaining 30% = 0.6 LST is shared only by the selected active public committee for that block, not by every registered wallet. Transaction fees go to the producer.

Emergency producers: prod-1 and prod-2 only keep the chain alive when no public miner is eligible. When public miners are active, they are not the passive 30% public committee.

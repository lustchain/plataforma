Exact order

Windows test flow

Keep it simple. One wallet, one clean folder, one package.

Download fresh ZIP

Download the Windows V31H2J Final Fix ZIP and extract it to a clean folder such as C:\LUST-Miner-H2J.

Register / Active

Open the register helper, claim faucet gas if needed, then send Register and Active / Heartbeat.

Start miner

Run START-LUST-MINER-WINDOWS.bat. It must show Chain ID 0x1b0b. If it shows 0x1 or block 0, stop.

Wait for sync

Let it sync naturally. Do not close while gap is falling. It should not reinstall the old snapshot just because gap is high.

Check status

Run STATUS-LUST-MINER-WINDOWS.bat. Good status: Chain ID 0x1b0b, peers above 0, gap falling or low.

Verify on-chain

Rewards must be confirmed by wallet balance, RPC or explorer. Blockscout may show native custom reward as 0.

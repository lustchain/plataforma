Step 2

Download the official miner.

The V7A21 miner restores the latest snapshot when needed and TX-FEED V3 pulls public pending transactions into the local miner node every 0.5 seconds. These downloads are active for the controlled external-miner test. Always verify the SHA256 before running the package. The package SHA changes when miner scripts or binaries change; the snapshot SHA changes when the snapshot updates.

Verified reward rules

Exactly how each 2 LST block reward is distributed.

The reward depends on how many public miners are active. The block producer is never counted again inside the public committee, and official/bootstrap addresses are excluded from the public 30% share.

0 public miners

Official emergency production.

Prod-1/prod-2 keep the chain alive as fallback. The official producer of the block receives the full 2 LST.

1 public miner

The public miner receives 2 LST.

The active public miner produces the block. Because there is no second public participant to form a committee, the same producer receives the full 2 LST.

2 or more public miners

70% producer · 30% public committee.

The block producer receives 1.4 LST. The other active public miners divide 0.6 LST. Official/bootstrap nodes receive none of this public committee share.

Transaction priority fees go to the actual producer of the block. The EIP-1559 base fee follows the network protocol and is not part of the 2 LST block reward.
TX-FEED V3

Fast transaction feed for public miners.

TX-FEED V3 checks the public pending transaction feed every 0.5 seconds and injects valid raw transactions into the local miner node. This gives public miners a real chance to include transactions when they produce the next block.

Windows

Two windows stay open.

START opens LUST MINER and LUST TX-FEED V3. The feed auto-detects local RPC port 18547 or 8545 and logs to LUST-Miner/logs/txfeed-v3.log.

Ubuntu / Linux

Background feed.

The Linux starter launches TX-FEED V3 in background. It uses local RPC 18547 and P2P 30313 by default to avoid conflicts with INRI or another geth. Check the live feed log at ~/LUST-Miner/logs/txfeed-v3.log.

Pending feed

Public endpoint.

The miner reads https://rpc.lustchain.org/pending-raw. This is the official public transaction feed used by TX-FEED V3.

Network safety

Official producer standby.

V7A21 prioritizes active public miners for normal block production. Official/bootstrap/prod-1/prod-2 remain available only as emergency fallback when public production stops, and they are excluded from the passive 30% public committee share while public miners are active.

Security

Never enter your seed.

The official site only asks MetaMask to sign/send a registration transaction. Never type your seed phrase on any website.

Wallet rule

One machine, one wallet.

For clean testing and production, use one registered wallet per miner machine. Do not run the same wallet on many devices at the same time.

Safe start

No isolated mining.

The miner starts with mining off, waits for peers, chain ID and gap, starts TX-FEED V3, then enables mining only when the node is ready.

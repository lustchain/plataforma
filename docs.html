LQC at a glance

The main difference of LUST Chain is not raw hardware mining. LQC is designed as community mining: active participation matters more than industrial hashrate.

One wallet, one chance

Each registered and active reward wallet represents one protocol mining identity. More CPU or GPU power does not multiply the chance of that same wallet.

Active heartbeat

Registered wallets must stay online, activated and recently active. Inactive, stale or offline wallets are not eligible for public mining rewards.

70/30 reward model

When public mining is active, the selected producer receives the producer share and the selected active public committee shares the public committee pool.

Fallback protection

If public miners go offline, the network is designed to keep producing blocks and route emergency bootstrap rewards to the Community Treasury path.

Not passive income

LQC mining is not staking and not a passive registration reward. Eligibility depends on active, synchronized and valid public miner operation.

Progressive decentralization

The early network uses official infrastructure for stability, while the public mining layer and treasury model move the system toward broader community participation.
